# 📚 The Ember Collection

A privacy-focused personal book library tracker. Track books you want to buy, books you're reading, and books you've finished - all stored locally on your device.

## ✨ Features

- 📖 Track books by status (Want to Buy, Unread, Reading, Finished)
- 🔥 Set priority levels for books you want to buy
- 📝 Add notes, quotes, and chapter notes
- ⭐ Rate books and track your favorites
- 🎯 Set and track reading goals
- 📊 View reading statistics and insights
- 📷 Barcode scanning for quick book adding
- 🌙 Dark mode support
- 📱 Works offline as a Progressive Web App

## 🔒 Privacy First

- **100% local storage** - all your data stays on your device in IndexedDB
- **No tracking** - zero analytics or cookies
- **No account required** - no login, no email collection
- **Works offline** - fully functional without internet after first load
- **Your data, your control** - export and import your library anytime

## 🛠️ Tech Stack

- Pure HTML/CSS/JavaScript (no frameworks)
- IndexedDB for local storage
- OpenLibrary API for book data
- QuaggaJS for barcode scanning
- Service Worker for offline support

---

**Note:** This repository contains only the app code. Your personal book collection data is stored locally on your device and never transmitted anywhere.
